# Stub files for VTK python

Stub files are required for VTK python autocomplete to work.

## How to use

Copy `*.ipy` files and replace `all.py`.

```bash
cp -rf typings/vtkmodules/* /<your-python-path>/site-packages/vtkmodules
```
